## Short Thesis

The majority of on-chain trading agents just suck. They either fail into making any profit or operate as black box nobody can really see through. 

What we need is an open source community driven quant agent. Nobody can be left out and everyone can contribute through either funding or developing.

The structure would be more or less like this:

- You can choose to be either a provider or developer.
- Providers stakes tokens inside a vault and earn shared profits from agent's operations.
- Developers can earn part of the vault funding + profits through verified repository contributions.

This create a community ecosystem where successful developing of the product is naturally incentivize.
